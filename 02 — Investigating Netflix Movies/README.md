# Investigating Netflix Movies and Guest Stars in The Office

## Project Overview
This project explores and analyzes Netflix shows data. The analysis includes data cleaning, visualization, and statistical analysis to uncover trends and insights related to these popular media sources.

## Project Files
- `notebook.ipynb`: The Jupyter notebook containing all the code, analysis, and visualizations for the project.

## Datasets
- The datasets used in this project were sourced from DataCamp and include information on Netflix shows.

## Key Concepts and Techniques
- **Data Cleaning:** Handling missing data and ensuring the datasets are prepared for analysis.
- **Data Visualization:** Using libraries like Matplotlib and Seaborn to create meaningful visual representations of the data.
- **Statistical Analysis:** Performing basic statistical tests and analyses to derive insights.

## Requirements
To run the Jupyter notebook, you’ll need the following Python libraries:
- Pandas
- NumPy
- Matplotlib
- Seaborn
