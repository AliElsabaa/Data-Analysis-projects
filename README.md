# 📊 Data Analysis Projects — DataCamp

A collection of exploratory data analysis projects completed as part of the 
**Data Scientist with Python** career track on DataCamp. Each project works 
with a real-world dataset to extract meaningful insights through analysis 
and visualization.

---

## 📁 Projects

### 01 — The GitHub History of the Scala Language
**Dataset:** ~30,000 commits from the official Scala GitHub repository  
**Goal:** Analyze the development history of the Scala programming language 
through its commit history and repository activity  
**Key Questions:**
- How has contributor activity changed over 10+ years?
- What does the commit history reveal about Scala's maturity as a language?

**Skills:** Time series analysis, commit data exploration, contributor patterns

---

### 02 — Investigating Netflix Movies
**Dataset:** Netflix content catalog (title, genre, country, release year, duration)  
**Goal:** Investigate whether Netflix movies are getting shorter over time  
**Key Findings:**
- Average movie duration dropped from 103 minutes (2011) to 90 minutes (2020)
- The apparent decline is largely explained by the rise of non-traditional genres 
  (Children, Documentaries, Stand-Up Comedy) — not feature films getting shorter
- Color-coded scatter plot reveals the genre composition driving the trend

**Skills:** Data filtering, scatter plot analysis, outlier investigation, 
genre-based color coding

---

### 03 — The World's Oldest Businesses
**Dataset:** Globally operating businesses with their founding dates and countries  
**Goal:** Discover which businesses have survived the longest and where they are located  
**Key Questions:**
- What are the oldest continuously operating businesses in the world?
- Which countries and regions have the highest concentration of ancient businesses?
- What patterns exist in business longevity across industries?

**Skills:** Geographic analysis, EDA, historical data exploration

---

### 04 — A Visual History of Nobel Prize Winners
**Dataset:** Nobel Prize winners dataset (category, year, country, gender)  
**Goal:** Visually explore trends in Nobel Prize awards across time, 
gender, and nationality  
**Key Questions:**
- Which countries dominate Nobel Prize wins?
- How has gender representation changed over time?
- Are there trends in awards by category across decades?

**Skills:** EDA, trend analysis, data visualization, historical pattern recognition
